Creates an evaluate.lua file for use with the ConstrainedOptimization UG4 plugin. To understand the use of an evaluate.lua file, consult the documentation of the ConstrainedOptimization UG4 plugin.

Parameters
[in]pathPath where the evaluate.lua file shall be written to.
[in]textbodyAuxillary text that shall be added into the evaluate.lua file
[in]names_of_constantsNames of variables or model parameters that are not optimized but have to be set
[in]values_of_constantsValues of aforementioned variables or model parameters that are not optimized but have to be set
[in]names_of_variablesNames of variables or model parameters that are optimized
[in]values_of_variablesValues of aforementioned variables or model parameters that are optimized
[in]names_of_initsNames of initial conditions or auxillary variables that are not optimized but have to be set
[in]values_of_initsValues of initial conditions or auxillary variables that are not optimized but have to be set
[in]grid_vecOptional parameter that is used to specify the cartesian coordinates of the grid if a PDE model is optimized
